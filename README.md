# 🚀 לוח דרושים - Jobs

ברוכים הבאים למאגר **Jobs** הרשמי של **AnLoMinus**.
מאגר זה מכיל לוח דרושים מבוסס Jekyll, המאפשר למעסיקים לפרסם משרות ולעובדים לחפש הזדמנויות בצורה פתוחה, מהירה וקלה.

---

## 🏗️ מבנה המאגר

```bash
/Jobs
│
├── _posts/                     # כל פוסט משרה הוא קובץ .md עם תאריך ושם המשרה
│   └── 2026-03-22-example-job.md
│
├── _config.yml                 # הגדרות Jekyll ל-GitHub Pages
├── index.md                    # דף הבית של לוח הדרושים
├── about.md                    # עמוד התמצית והאודות
├── Gemfile                     # תלויות ההרצה (github-pages)
└── README.md                   # תיאור המאגר (קובץ זה)
```

---

## 🪶 איך לפרסם משרה?

כדי להוסיף משרה חדשה:
1. שכפלו (Fork) את המאגר או פתחו ענף (Branch) חדש (אם יש לכם הרשאות).
2. תחת התיקייה `_posts/`, צרו קובץ חדש בפורמט: `YYYY-MM-DD-job-title.md`.
3. השתמשו בתבנית הבאה:

```markdown
---
layout: post
title: "מעצב/ת UI/UX (דוגמה)"
categories: "עיצוב"
---

## תיאור המשרה
[פרטו כאן על המשרה...]

## דרישות
- 
- 

## תנאים ומיקום
- 

## איך להגיש מועמדות
יש לשלוח קורות חיים ל-...
```
4. שמרו ופתחו **Pull Request** אל המאגר הראשי.

## 🌐 הפעלה (GitHub Pages)

האתר מוגדר לפעול באופן אוטומטי כשמעלים אותו לענף הראשי דרך **GitHub Pages**.
בקרו בכתובת המיועדת: **[https://anlominus.github.io/Jobs/](https://anlominus.github.io/Jobs/)**

כדי להריץ לבדיקה מקומית (Local):
```bash
bundle install
bundle exec jekyll serve
```
